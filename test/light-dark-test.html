Usage

Initialize the SpawnKit theme toggle component:

const toggle = new SpawnKitThemeToggle(containerElement, { defaultMode: 'light', // 'light' | 'dark' | 'auto' onToggle: (mode) => { console.log('Theme changed to:', mode); }, persist: true // save to localStorage }); // Programmatic control toggle.setMode('dark'); toggle.getMode(); // ‚Üí 'dark'
