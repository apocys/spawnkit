## Summary
**3 QA sessions** executed with deep methodology (qa-tester skill). **Result: NOT demo-ready.**

## 🔴 CRITICAL BUGS

### BUG-001: Overlay Stacking Chaos (12+ panels open simultaneously)
- **Root cause**: `closeAllPanels()` in `app-init.js:2116` is incomplete — missing `closeMissionsPanel`, `closeSettingsPanel`, `closeChatHistory`, and orchestration close. Also, `openMailbox()` and `openDetailPanel()` don't call `closeAllPanels()` at all — they cherry-pick individual closes.
- **Impact**: First-time user sees 10+ overlapping dialogs. Interface becomes unusable.
- **Fix**: Make `closeAllPanels()` comprehensive and call it at the start of every `open*` function.

### BUG-002: Loading States Never Resolve
- **Affected panels**: Cron Jobs ("Loading cron jobs…"), Memory ("Loading memory…"), Missions ("Loading missions…"), Settings ("Loading settings…"), Remote Offices ("Connecting to fleet relay…"), Chat History ("Loading chat history...")
- **Root cause**: Panels open on page load (overlay stacking) before data-bridge has finished connecting. The `render*` functions are called when user opens the panel, but since panels are already "open" from boot, they show stale loading states.
- **Fix**: Ensure panels only open via user action (after BUG-001 fix, this resolves itself).

### BUG-003: Agent Creation Wizard Hangs on Step 3 (Model Selection)
- **Steps**: Add Agent → Fill name → Select role → Reach model selection → Click Create
- **Root cause**: All API keys marked as "Not set" — wizard provides no error or fallback.
- **Impact**: No new agents can be created. Dead-end UX for new users.
- **Fix**: Show clear error about missing API keys, or provide demo mode with simulated agents.

### BUG-004: Shell Flicker (Executive → Mission Desk Double-Render)
- **Root cause**: `index.html` renders full Executive shell first, then `mission-desk.js:init()` injects Mission Desk and hides legacy chrome (grid, statusbar). This causes a visible flicker on load.
- **Fix**: Make Mission Desk the default shell; remove or conditionalize the legacy Executive chrome.
