# Changelog

All notable changes to the SpawnKit SimCity Theme.

## [1.0.0] — 2026-02-22

### Added
- Initial release
- Full 20×20 isometric tile map with Kenney Nature Kit sprites
- 6 animated agent characters (8-frame idle + walk, 4 directions)
- BFS pathfinding on road network
- Smart behavior system: working, campfire, visiting, meeting, returning
- Context-aware speech bubbles (work, social, collaboration)
- Interactive canvas with pan, zoom, and reset controls
- Agent detail panel with metrics, skills, and TODO lists
- CEO chat panel with multi-agent routing
- Missions panel for active task tracking
- Crons panel for scheduled job management
- Memory panel with golden rules and daily logs
- Settings panel with preferences
- Status bar with village stats, clock, and command input
- Loading screen with progress bar
- Toast notification system
- Demo data fallback when no backend connected
- Map generation with forests, paths, campfire area, and flower gardens
